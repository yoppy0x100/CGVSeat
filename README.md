# CGVSeat

adalah tools yang dibuat untuk melihat ketersediaan kursi di cgv cinema

## Installation

This tool is available 2 kind of way,
used directly or used as api library

**Git** - _directly use_

```bash
  git clone https://github.com/yoppy0x100/CGVSeat
  cd CGVSeat
  composer install
  php run.php
```
